# Volt React Frontend

## Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on http://localhost:3000

## Backend

Make sure the Flask API is running:

```bash
cd ..
python api.py
```

Backend API runs on http://localhost:5000
